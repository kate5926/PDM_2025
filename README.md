# Resumen de Prácticas del Curso

Este repositorio contiene un conjunto de prácticas desarrolladas en Kotlin y Flutter, que abarcan desde conceptos básicos de programación hasta el desarrollo de aplicaciones para Android.

## Índice de Prácticas

A continuación se presenta un resumen de cada una de las prácticas incluidas en este repositorio.

---

### [Práctica 1: Fundamentos de Kotlin](./Practica_1)

Esta práctica se enfoca en los conceptos básicos de Kotlin a través de cuatro ejercicios interactivos: evaluación de empleados, piedra-papel-tijera, una calculadora elemental y un juego de adivinar el número.

---

### [Práctica 2: Programación Orientada a Objetos](./Practica_2/Ejercicios/)

Esta sección se centra en la aplicación de los principios de la Programación Orientada a Objetos (POO) en Kotlin, con ejercicios como la simulación de una cuenta bancaria, cálculo de descuentos, modelado de figuras geométricas y un sistema de biblioteca.

---

### [Práctica 3: Desarrollo de Aplicaciones Android](./Practica_3/)

Esta práctica contiene dos proyectos independientes de Android Studio, diseñados para introducir al desarrollo de aplicaciones móviles: una imagen interactiva y un reproductor de música simple.

---

### [Práctica 4: Comunicación entre Actividades](./Practica_4/)

Esta práctica explora la comunicación entre actividades en Android mediante el uso de `Intent`, `Serializable` y el manejo de resultados. Incluye dos mini-aplicaciones: un editor de perfil y un editor de notas.

---

### [Práctica de Clase: Ciclo de Vida de una Activity](./Practica-clase/)

Este ejercicio práctico demuestra el ciclo de vida de una `Activity` en Android. La aplicación incluye un contador simple y muestra `Toasts` y logs en cada cambio de estado del ciclo de vida para facilitar la observación.

---

### [Práctica 5: Fragments y Navegación](./Practica_5/)

Este proyecto implementa un flujo de pedido de comida utilizando Fragments y el componente de Navegación en Android. El usuario puede armar su pedido paso a paso, seleccionando la comida principal, los extras, y revisando un resumen.

---

### [Práctica con RecyclerView](./Practica_RecyclerView/)

Esta práctica implementa un sistema completo de gestión de usuarios utilizando `RecyclerView` en Kotlin. La aplicación permite realizar operaciones CRUD mediante eventos táctiles y diálogos personalizados.

---

### [Práctica 6: Interfaces Estáticas con Flutter](./practica_6/)

Este proyecto es una aplicación Flutter básica que presenta tres interfaces estáticas (inicio, perfil y hobbies) para practicar el uso de widgets de layout.

---

### [Práctica de Clase 2: Navegación con BottomNavigationBar](./practica_clase_2/)

Una aplicación Flutter que demuestra el uso de `BottomNavigationBar` para la navegación entre diferentes pantallas, incluyendo un ejemplo de paso de datos para editar un perfil de usuario.

---

### [Práctica de Clase 3: CRUD con Provider](./practica_clase_3/)

Implementa un sistema completo de gestión de usuarios usando Flutter, Provider y el patrón MVVM. Incluye CRUD, validaciones y filtro dinámico.

---

### [Práctica: Pantalla de Login con Flutter](./practica_pantalla_login/)

Este proyecto implementa una pantalla de inicio de sesión con validación de campos para correo electrónico y contraseña en Flutter.

---

### [Examen Parcial: Juego de Colores](./Examen_Parcial/)

Aplicación móvil desarrollada en Kotlin para Android que implementa un juego interactivo donde el usuario debe identificar y seleccionar colores coincidentes bajo un límite de tiempo.
